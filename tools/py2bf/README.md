# Python subset -> Brainfuck compiler (educational)

Компилятор очень ограниченного подмножества Python в Brainfuck.

## Поддержка
- Переменные и целочисленные значения
- Присваивания:
  - `a = 3`
  - `b = a`
  - `c = a + 2`
  - `d = a - b`
- `while <var>:` (условие: переменная != 0)
- `if <var>:` (условие: переменная != 0)
- Вложенные циклы/условия

## Адресация переменных в ленте
- Каждой переменной выделяется фиксированная ячейка BF ленты (`cell 0..N-1`).
- Дополнительно компилятор резервирует временные ячейки `temp0`, `temp1`.

## Оптимизация
Peephole-оптимизация Brainfuck:
- удаление нейтральных пар `<>`, `><`, `+-`, `-+`
- удаление пустых циклов `[]`
- схлопывание `[-][-]` -> `[-]`

## Запуск
```bash
python -m tools.py2bf.main program.py -o program.bf
```


Примечание: `if var:` в текущей реализации выполняет тело один раз и обнуляет `var`.
